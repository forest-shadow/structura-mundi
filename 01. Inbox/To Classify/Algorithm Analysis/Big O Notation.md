# Big O Notation

## Краткое определение

`Big O Notation` — это способ задавать верхнюю асимптотическую оценку роста вычислительных затрат при увеличении размера входа.

## Роль в ветке

Это общий аналитический инструмент для секции `[[Algorithm Analysis]]`, а не тема только про сортировки.
