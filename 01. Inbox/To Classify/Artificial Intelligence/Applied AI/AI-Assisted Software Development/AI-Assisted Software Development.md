# AI-Assisted Software Development

## Краткое определение области

`AI-Assisted Software Development` — это обзорная ветка о программной инженерии, в которой значимая часть анализа, проектирования, модификации кода и инженерной координации делегируется ИИ-системам и агентам, способным работать с репозиторием, инструкциями, инструментами и проверяемыми артефактами.

## Что входит в эту тему

- `[[AI-Assisted Programming]]`

## Как устроена ветка

- `AI-Assisted Software Development` теперь фиксирует именно прикладной сценарий внутри более широкой ветки `[[Applied AI]]`, а не весь AI-контекст целиком.
- Общие агентные механизмы и протоколы вынесены в соседнюю ветку `[[AI Agent Systems]]`.
- `[[OpenAI Codex]]` остается важным частным примером agentic-инструмента для software engineering, но не исчерпывает саму тему.
- На текущем этапе внутри этой ветки уже выделяется дочерняя статья `[[AI-Assisted Programming]]`; остальные аспекты можно добавлять позже как отдельные статьи, а не смешивать сразу в корневой overview.

## Рекомендуемый маршрут чтения

1. Начать с `[[Artificial Intelligence]]`.
2. Затем перейти в `[[Applied AI]]`.
3. При необходимости сначала зафиксировать инфраструктурный контекст через `[[AI Agent Systems]]` и `[[OpenAI Codex]]`.
4. После этого перейти к `[[AI-Assisted Software Development]]`.
5. Завершить чтением `[[AI-Assisted Programming]]`.
