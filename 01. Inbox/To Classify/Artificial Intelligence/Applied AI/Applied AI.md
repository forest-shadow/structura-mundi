# Applied AI

## Краткое определение области

`Applied AI` — это обзорная ветка о практических сценариях, в которых ИИ используется как рабочий инструмент для выполнения содержательных задач, а не как объект изучения сам по себе.

## Что входит в эту тему

- `[[AI-Assisted Software Development]]`
- `[[AI-Assisted Project Management]]`
- `[[AI-Assisted Personal Effectiveness]]`

## Как устроена ветка

- Ветка строится по сценариям применения, а не по конкретным продуктам.
- Общие агентные механизмы и протоколы вынесены в соседнюю ветку `[[AI Agent Systems]]`.
- `AI-Assisted Software Development` здесь является отдельным прикладным кластером, а не корневой рамкой для всех AI-тем.

## Рекомендуемый маршрут чтения

1. Начать с `[[Applied AI]]`.
2. Затем выбрать прикладной сценарий: software development, project management или personal effectiveness.
3. Если для понимания не хватает инфраструктурного слоя, перейти в `[[AI Agent Systems]]`.
