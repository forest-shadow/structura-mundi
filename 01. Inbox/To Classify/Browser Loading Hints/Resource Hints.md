# Resource Hints

## Краткое определение области

Обзорная заметка для hints, которыми разработчик подсказывает браузеру, какие ресурсы стоит загрузить заранее и с каким ожиданием дальнейшего использования.

## Что входит в эту тему

- `[[Preload]]`
- `[[Prefetch]]`
- `[[Modulepreload]]`

## Как устроена ветка

- `Resource Hints` собирает механизмы ранней и упреждающей загрузки ресурсов.
- Пока сюда входят только наиболее устойчивые и часто обсуждаемые узлы.
- Более специализированные hints можно добавить позже, если ветка вырастет.

## Рекомендуемый маршрут чтения

1. Начать с общей идеи `Resource Hints`.
2. Затем читать `[[Preload]]`.
3. После этого читать `[[Prefetch]]`.
4. Затем перейти к `[[Modulepreload]]`.
