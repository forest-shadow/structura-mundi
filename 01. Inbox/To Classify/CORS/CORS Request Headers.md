# CORS Request Headers

## Краткое определение

Каноническая статья про request-headers, через которые браузер описывает origin и ожидаемые свойства cross-origin запроса.

## Основная идея или механизм

Эта заметка должна объяснять, какие данные браузер отправляет серверу для CORS-проверки и как эти заголовки участвуют в preflight и основном запросе.

## Границы темы

- Входит: роль request-headers в CORS-механизме.
- Не входит: полный список server-side response-решений.
