# CSS

## Краткое определение области

Обзорная заметка для языка CSS как системы декларативного описания визуального представления документа.

## Что входит в эту тему

- `[[CSS Cascade]]`

## Как устроена ветка

- `CSS` служит верхнеуровневой рамкой для заметок про язык CSS.
- Пока внутрь ветки вынесен только кластер `CSS Cascade`, потому что именно в нём естественно живут `CSS Specificity` и остальные механизмы каскада.
- Другие подветки вроде layout, selectors или units стоит добавлять только по мере появления самостоятельного корпуса.

## Рекомендуемый маршрут чтения

1. Начать с общей идеи `CSS`.
2. Затем перейти к `[[CSS Cascade]]`.
3. После этого читать `[[CSS Specificity]]`.
4. Затем перейти к `[[CSS Cascade Origins, Importance, Layers, and Order]]`.

## Соседние overview-ветки

- Позже рядом могут появиться подветки про selectors, layout или responsive design, но сейчас они не нужны для минимальной структуры.
