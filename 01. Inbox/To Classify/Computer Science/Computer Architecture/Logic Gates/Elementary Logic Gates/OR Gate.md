# OR Gate

## Краткое определение

`OR Gate` - это логический элемент, который выдает `1`, если хотя бы один из входов имеет значение `1`.

## Основная идея или механизм

`OR Gate` реализует аппаратную форму логической дизъюнкции. Для двух входов ложный результат возникает только тогда, когда оба входа равны `0`.

## Границы темы

- В тему входит базовая операция выбора по условию "хотя бы одно из".
- С ней соседствует `[[Logical Disjunction]]`, но та заметка описывает формально-логическую операцию, а не gate-level реализацию.

## Примеры, случаи или следствия

- Объединение нескольких сигналов события в один выход.
- Формирование условия запуска при наличии любого из допустимых входов.
- Получение `NOR` как инверсии результата `OR`.
