# Frontend Engineering

## Краткое определение области

`Frontend Engineering` — это доменная обзорная заметка для веток про браузерную платформу, пользовательский интерфейс, модели рендеринга и прикладные framework-level abstractions.

## Что входит в эту тему

- `[[React]]`
- `[[CSS]]`
- `[[Browser Page Processing]]`
- `[[Browser Rendering]]`
- `[[CORS]]`

## Как устроена ветка

- `React` собирает framework-level UI branch.
- `CSS` собирает language-level styling branch.
- `Browser Page Processing`, `Browser Rendering` и `CORS` описывают browser-platform mechanisms, которые лежат ниже application framework уровня.

## Рекомендуемый маршрут чтения

1. Для browser fundamentals идти в `[[Browser Page Processing]]`, `[[Browser Rendering]]` и `[[CORS]]`.
2. Для UI framework branch идти в `[[React]]`.
3. Для styling model идти в `[[CSS]]`.
