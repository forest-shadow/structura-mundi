# Batching and Request Coalescing

## Краткое определение области

`Batching and Request Coalescing` — это обзорная ветка про объединение мелких операций или одинаковых запросов, чтобы уменьшить накладные расходы на сеть, storage, locking и downstream calls.

## Что входит в эту тему

- request coalescing
- micro-batching
- bulk APIs
- write batching
- duplicate suppression
- latency-throughput trade-offs
