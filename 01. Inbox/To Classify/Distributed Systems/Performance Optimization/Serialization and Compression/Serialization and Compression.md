# Serialization and Compression

## Краткое определение области

`Serialization and Compression` — это обзорная ветка про формат, размер и стоимость кодирования данных, передаваемых между сервисами, очередями, кэшами и storage layers.

## Что входит в эту тему

- binary formats
- schema evolution cost
- compression ratio
- CPU versus network trade-off
- payload size
- streaming serialization
