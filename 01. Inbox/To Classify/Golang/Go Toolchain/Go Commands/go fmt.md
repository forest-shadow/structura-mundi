# go fmt

## Краткое определение

`go fmt` - это команда Go для автоматического форматирования исходного кода по стандартным правилам экосистемы.

## Основная идея или механизм

Команда нормализует стиль исходников через единый formatter, уменьшая пространство для stylistic divergence между проектами и разработчиками.

## Границы темы

- В тему входит автоматическое форматирование и его роль в developer workflow.
- Рядом находится более широкая тема idiomatic style, но она не сводится только к formatter output.
