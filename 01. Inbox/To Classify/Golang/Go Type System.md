# Go Type System

## Краткое определение

Статья о типовой модели Go: базовых типах, структурах, указателях, методах и общей строгости типизации.

## Основная идея или механизм

Type system в Go сочетает простоту синтаксиса с достаточно строгими правилами совместимости и явным контролем над формами данных.
