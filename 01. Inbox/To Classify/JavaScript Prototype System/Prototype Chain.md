# Prototype Chain

## Краткое определение

Каноническая статья для модели поиска свойств и наследования в JavaScript через последовательность прототипов.

## Основная идея или механизм

Нужно объяснить, как объект, его внутренний прототип и прототипы выше по цепочке образуют механизм разрешения свойств.
