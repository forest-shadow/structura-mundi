# JavaScript Scope Model

## Краткое определение области

Обзорная заметка про то, как JavaScript определяет область видимости имен, разрешает обращения к переменным и почему на этой основе возможны closures.

- `[[Scope Chain]]`

## Как устроена ветка

- `JavaScript Scope Model` служит обзорной рамкой для тем про scope и closure.
- `Lexical Scope` объясняет статическую основу модели.
- `Scope Chain` объясняет разрешение имен при выполнении.
- `JavaScript Closure` описывает сохранение доступа к внешнему лексическому окружению.
