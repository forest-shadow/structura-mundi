# Lexical Scope

## Краткое определение

Каноническая статья про lexical scope как правило, по которому область видимости имен в JavaScript определяется структурой исходного кода, а не порядком вызовов во время выполнения.

## Основная идея или механизм

Эта заметка должна объяснять, почему функции и блоки формируют лексические границы видимости и как именно это создаёт основу для closure.

## Границы темы

- Входит: статическое определение области видимости, вложенные области, связь с функциями.
- Не входит: вся тема runtime resolution и сохранения окружения после выхода из функции.
