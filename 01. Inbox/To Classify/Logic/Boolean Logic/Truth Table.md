# Truth Table

## Краткое определение

`Truth Table` - это табличный способ перечислить все возможные комбинации значений истинности входящих высказываний и показать, какое значение принимает результирующая формула.

## Основная идея или механизм

Таблица истинности делает булеву формулу полностью явной: вместо интуитивного рассуждения она показывает поведение формулы для каждого допустимого набора входных значений. Поэтому через truth tables удобно проверять эквиваленции, следование и отличие одной связки от другой.

## Границы темы

- В тему входит сама техника полного перебора значений истинности.
- Рядом находится вопрос о смысле отдельных связок, но он естественнее раскрывается в `[[Logical Connectives]]`.

## Примеры, случаи или следствия

- Для `Logical Negation` таблица показывает инверсию значения.
- Для `Logical Conjunction` таблица показывает, что результат истинен только при двух истинных компонентах.
- Для `Logical Equivalence` таблица позволяет быстро увидеть, совпадают ли значения двух формул во всех строках.
