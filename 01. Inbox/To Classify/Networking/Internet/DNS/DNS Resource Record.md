# DNS Resource Record

## Краткое определение

`DNS Resource Record` — это базовая единица данных в DNS, фиксирующая имя, тип записи, класс и связанное с ними значение.

## Основная идея или механизм

Именно resource records составляют содержимое DNS-зон и являются теми данными, которые в итоге получают клиенты и резолверы. Через разные типы записей DNS выражает адресацию, алиасы, почтовую маршрутизацию и другие формы именования.

## Границы темы

- Входит: ресурсные записи как элемент модели данных DNS и их роль в ответах DNS.
- Не входит: отдельный подробный разбор всех record types по одному.

## Примеры, случаи или следствия

- A- и AAAA-записи связывают имя с IP-адресом.
- CNAME-подобные записи позволяют выразить каноническое имя вместо прямого адресного значения.
