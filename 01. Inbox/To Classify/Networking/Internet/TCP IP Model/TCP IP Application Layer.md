# TCP IP Application Layer

## Краткое определение

Каноническая статья про application layer в модели TCP/IP как верхний слой, объединяющий прикладные сетевые протоколы и функции представления/сеанса в более практической стековой форме.

## Основная идея или механизм

Эта заметка должна объяснять, что application layer в TCP/IP покрывает более широкий верхний диапазон функций, чем application layer в OSI, потому что практическая модель не выделяет отдельные session и presentation уровни.

## Границы темы

- Входит: верхний прикладной слой TCP/IP и его общая роль в стеке.
- Не входит: вся бизнес-логика приложения вне сетевого взаимодействия.
