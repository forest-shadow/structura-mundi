# TCP IP Internet Layer

## Краткое определение

Каноническая статья про internet layer как слой TCP/IP, отвечающий за межсетевую адресацию и доставку пакетов между сетями.

## Основная идея или механизм

Эта заметка должна объяснять, как internet layer обеспечивает routing и internetworking в практическом Internet-стеке.

## Границы темы

- Входит: адресация и межсетевой перенос пакетов.
- Не входит: end-to-end управление транспортом между приложениями.
