# TCP IP Model

## Краткое определение области

Обзорная заметка для `TCP/IP Model` как практической многослойной модели сетевого стека, на которой основаны современные Internet-протоколы.

## Что входит в эту тему

- `[[TCP IP Link Layer]]`
- `[[TCP IP Internet Layer]]`
- `[[TCP IP Transport Layer]]`
- `[[TCP IP Application Layer]]`

## Как устроена ветка

- `TCP IP Model` служит обзорной рамкой для слоёв Internet protocol suite.
- Для двусмысленных слоёв используются минимально уточнённые канонические имена, чтобы не конфликтовать с веткой `OSI Model`.
- Более мелкие сравнительные и протокольные темы пока не выносятся в отдельные узлы.

## Рекомендуемый маршрут чтения

1. Начать с общей идеи `TCP IP Model`.
2. Затем идти по слоям снизу вверх: `[[TCP IP Link Layer]]`, `[[TCP IP Internet Layer]]`, `[[TCP IP Transport Layer]]`, `[[TCP IP Application Layer]]`.
3. После этого при необходимости смотреть `[[OSI Model]]` как соседнюю концептуальную модель.
