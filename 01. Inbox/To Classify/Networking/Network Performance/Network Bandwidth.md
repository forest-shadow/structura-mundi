# Network Bandwidth

## Краткое определение

`Network Bandwidth` — это статья о верхней границе объема данных, который канал или соединение может передать за единицу времени.

## Границы темы

- Входит: bandwidth как capacity metric в сети.
- Не входит: полный разбор всех практик измерения, congestion control и application-level performance tuning.

## Связи с другими заметками

Родительская тема:

`[[Network Performance]]`

Связанные заметки:

- `[[Network Throughput]]`
- `[[Network Latency]]`

## Примеры, случаи или следствия

- Высокая bandwidth сама по себе не гарантирует высокую реальную скорость передачи полезных данных.
- Пропускная способность соединения должна рассматриваться вместе с latency, loss и overhead.
