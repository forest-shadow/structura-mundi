# Network Latency

## Краткое определение

`Network Latency` — это статья о задержке между отправкой данных и их доставкой или наблюдаемым ответом сети.

## Границы темы

- Входит: latency как time-based network metric.
- Не входит: полный разбор всех аппаратных и программных источников задержки.

- `[[Network Jitter]]`

## Примеры, случаи или следствия

- Низкая latency важна не только для интерактивных приложений, но и для эффективности некоторых протоколов передачи.
- Большая bandwidth не компенсирует автоматически высокую latency.
