# Link Layer Protocols

## Краткое определение области

`Link Layer Protocols` — это обзорная заметка о протоколах, которые обеспечивают передачу данных по локальной среде и взаимодействие узлов в пределах одного канального сегмента.

## Что входит в эту тему

- Базовая роль link-layer protocols в сетевом стеке.
- Связь с локальной средой передачи и канальным взаимодействием.
- Сопоставление с `[[TCP IP Link Layer]]` и `[[Data Link Layer]]`.

## Как устроена ветка

- Пока это overview-note без вынесенных child-статей.
- Конкретные протоколы вроде `Ethernet`, `ARP` и `PPP` стоит добавлять позже, когда появится плотный сравнительный корпус.
