# Network Layer Protocols

## Краткое определение области

`Network Layer Protocols` — это обзорная заметка о протоколах, которые поддерживают межсетевую адресацию, доставку пакетов между сетями и связанное с этим управление.

## Что входит в эту тему

- Общая роль network-layer protocols.
- Связь с адресацией и межсетевой доставкой.
- `[[Routing Protocols]]` как вложенный подкласс протоколов управления маршрутами.

## Как устроена ветка

- `Network Layer Protocols` собирает общую рамку для протоколов inter-network delivery.
- `Routing Protocols` вынесены в отдельную вложенную ветку, потому что это устойчивый control-oriented подкласс.
- Более узкие notes про `IP`, `ICMP` и другие network-layer protocols пока не создаются.
