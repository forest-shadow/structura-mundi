# Routing Protocols

## Краткое определение области

`Routing Protocols` — это обзорная заметка о протоколах, через которые маршрутизаторы и другие сетевые узлы обмениваются маршрутной информацией и согласуют пути доставки трафика.

## Что входит в эту тему

- Routing protocols как особый подкласс network-layer control protocols.
- Связь с общей темой `[[Routing]]`.
- Роль маршрутной информации в выборе путей между сетями.

- Пока это overview-note без вынесенных child-статей.
