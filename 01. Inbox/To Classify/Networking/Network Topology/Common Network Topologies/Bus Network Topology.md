# Bus Network Topology

## Краткое определение

`Bus Network Topology` — это статья о topology pattern, в котором несколько узлов подключены к общей линии передачи.

## Границы темы

- Входит: bus как общая схема организации сети на общем канале.
- Не входит: полный исторический разбор конкретных Ethernet-реализаций.

## Примеры, случаи или следствия

- Общая среда передачи делает тему коллизий и координации доступа особенно важной.
- Отказ или деградация общей линии может затрагивать сразу всю сеть.
