# Hybrid Network Topology

## Краткое определение

`Hybrid Network Topology` — это статья о topology pattern, в котором сеть сочетает несколько базовых topology forms вместо одного чистого шаблона.

## Границы темы

- Входит: hybrid topology как общая рамка для смешанных сетевых схем.
- Не входит: полный каталог всех реальных сетевых архитектур.

## Примеры, случаи или следствия

- Реальные сети часто комбинируют локально разные patterns ради баланса между стоимостью, масштабируемостью и резервированием.
- Такой узел полезен как напоминание, что канонические topology names работают как модели, а не как исчерпывающий каталог реальности.
