# Mesh Network Topology

## Краткое определение

`Mesh Network Topology` — это статья о topology pattern, в котором узлы соединены множеством прямых связей и сеть получает альтернативные пути передачи.

## Границы темы

- Входит: mesh как схема высокой связности и резервирования.
- Не входит: полный разбор routing protocols и всех wireless mesh implementations.
