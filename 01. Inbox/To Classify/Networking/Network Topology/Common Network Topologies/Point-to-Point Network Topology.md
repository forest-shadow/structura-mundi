# Point-to-Point Network Topology

## Краткое определение

`Point-to-Point Network Topology` — это статья о topology pattern, в котором два узла соединены прямым каналом связи.

## Границы темы

- Входит: point-to-point как простейшая схема сетевого соединения.
- Не входит: полный разбор всех протоколов и физических сред, которые могут использовать такую связь.

## Примеры, случаи или следствия

- Прямая связь между двумя устройствами упрощает адресацию и контроль пути.
- Такой pattern не дает встроенной многоточечной масштабируемости без дополнительных узлов и связей.
