# Ring Network Topology

## Краткое определение

`Ring Network Topology` — это статья о topology pattern, в котором узлы соединены в замкнутую цепь.

## Границы темы

- Входит: ring как схема циклической связности между узлами.
- Не входит: полный разбор всех token-based и optical ring technologies.

## Примеры, случаи или следствия

- Замкнутый путь задает предсказуемую структуру передачи между соседними узлами.
- Отказ одного сегмента может сильно влиять на связность, если в схеме нет резервирования.
