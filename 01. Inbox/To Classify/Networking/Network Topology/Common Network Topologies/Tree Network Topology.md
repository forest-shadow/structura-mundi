# Tree Network Topology

## Краткое определение

`Tree Network Topology` — это статья о topology pattern, в котором сеть организована иерархически через уровни ветвления от более центральных узлов к периферийным.

## Границы темы

- Входит: tree как иерархическая схема организации сети.
- Не входит: полный разбор всех enterprise и datacenter hierarchies.

## Примеры, случаи или следствия

- Иерархия упрощает масштабирование и организацию больших сетей.
- Верхние уровни дерева могут становиться критическими точками нагрузки и отказа.
