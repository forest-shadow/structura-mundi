# TCP IP Link Layer

## Краткое определение

Каноническая статья про link layer в модели TCP/IP как слой, отвечающий за доставку данных по локальной среде передачи и взаимодействие с физической сетью.

## Основная идея или механизм

Эта заметка должна объяснять, что link layer в TCP/IP объединяет функции, которые в модели OSI обычно разделяются между physical и data link уровнями.

## Границы темы

- Входит: локальная передача по сети и связь стека с физической средой.
- Не входит: межсетевой маршрут и логическая адресация Internet layer.
