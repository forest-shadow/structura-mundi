# TCP IP Transport Layer

## Краткое определение

Каноническая статья про transport layer в модели TCP/IP как слой, отвечающий за end-to-end передачу данных между прикладными процессами.

## Основная идея или механизм

Эта заметка должна объяснять, как transport layer в TCP/IP работает поверх internet layer и чем он practically соотносится с transport layer в OSI.

## Границы темы

- Входит: end-to-end transport between endpoints.
- Не входит: прикладная семантика верхнего слоя и логика конкретных приложений.
