# OSI Model

## Краткое определение области

Обзорная заметка для `OSI Model` как концептуальной семиуровневой модели, через которую описывают функции сетевого взаимодействия.

## Как устроена ветка

- `OSI Model` служит обзорной рамкой для семи уровней сетевой модели.
- Каждый слой вынесен в отдельную `article`, потому что у него есть собственная устойчивая роль.
- Более мелкие темы пока не выносятся в дополнительные ветки, чтобы не усложнять структуру раньше времени.
