# Functional Components

## Краткое определение

Статья о функциональных компонентах как основной современной форме определения компонентов в React.

## Основная идея или механизм

Функциональный компонент описывает UI через функцию от входных данных и связывается с логикой состояния и эффектов через hooks.

## Границы темы

- В тему входит форма компонента как функции и её роль в современной модели React.
- На границе находятся hooks, class components и общая модель рендеринга.
