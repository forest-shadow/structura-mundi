# JSX

## Краткое определение

JSX - это синтаксическое расширение JavaScript, которое позволяет декларативно описывать структуру пользовательского интерфейса в React.

## Основная идея или механизм

Нужно раскрыть, как JSX связывает описание UI с компонентной моделью React и почему он не является HTML в буквальном смысле.

## Границы темы

- Что относится к JSX.
- Чем JSX отличается от шаблонного языка или прямого вызова React API.
