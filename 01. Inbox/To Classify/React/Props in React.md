# Props in React

## Краткое определение

Статья о `props` как механизме передачи входных данных в React-компоненты.

## Основная идея или механизм

`Props` задают внешний интерфейс компонента и определяют, какие данные приходят в него сверху, не превращая компонент автоматически в источник собственного состояния.

## Границы темы

- В тему входит форма входных данных компонента и их роль в композиции.
- На границе находятся `state`, `context` и более общая тема рендеринга.
