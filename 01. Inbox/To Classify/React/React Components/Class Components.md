# Class Components

## Краткое определение

Статья о классовых компонентах как исторически важной, но в основном legacy-форме определения компонентов в React.

## Основная идея или механизм

Классовый компонент связывает rendering, state и lifecycle-методы через объектную форму компонента, отличающуюся от современной hook-based модели.

## Границы темы

- В тему входит class-based форма React-компонента и её место в развитии React.
- На границе находятся lifecycle methods и сравнение с функциональными компонентами.
