# React Hooks

## Краткое определение области

React Hooks - это механизм React, который позволяет использовать состояние, эффекты, контекст и другие возможности React внутри функциональных компонентов и custom hooks.

## Что входит в эту тему

- правила использования hooks;
- базовые встроенные hooks;
- паттерн создания custom hooks;
- границы между локальным состоянием, побочными эффектами и повторным использованием логики.
