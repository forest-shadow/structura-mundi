# React Rendering Model

## Краткое определение

React Rendering Model - это способ, которым React вычисляет, когда и как пересчитывать компонентное дерево и обновлять представление интерфейса.

## Основная идея или механизм

Нужно раскрыть, как связаны render, reconciliation, state updates и повторные рендеры компонентов.

## Границы темы

- Что относится к rendering model.
- Чем модель рендеринга отличается от бизнес-логики компонентов или отдельных hooks.
