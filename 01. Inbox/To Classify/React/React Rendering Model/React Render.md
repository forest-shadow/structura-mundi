# React Render

## Краткое определение

Каноническая статья про render в React как фазу вычисления нового описания UI на основе текущих props, state и контекста.

## Основная идея или механизм

Эта заметка должна объяснять, что render в React не равен прямой перерисовке DOM, а относится к вычислению следующего представления интерфейса.

## Границы темы

- Входит: render как часть модели обновления React.
- Не входит: полный разбор reconciliation и commit-like последствий.
