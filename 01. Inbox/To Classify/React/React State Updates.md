# React State Updates

## Краткое определение

Каноническая статья про state updates в React как события изменения данных компонента, которые инициируют пересчёт UI.

## Основная идея или механизм

Эта заметка должна объяснять, как обновления состояния становятся входом в rendering model и почему их полезно рассматривать отдельно от конкретных hook API.

## Границы темы

- Входит: state updates как часть общей модели обновления.
- Не входит: полный разбор конкретных hooks кроме связи с ними.
