# React

## Краткое определение области

React - это библиотека для построения пользовательских интерфейсов через декларативное описание компонентов, состояния и процесса рендеринга.

## Что входит в эту тему

- JSX как форма описания UI;
- компоненты и передача данных через props;
- модель рендеринга и обновления интерфейса;
- hooks как способ выразить состояние, эффекты и повторно используемую логику.

## Как устроена ветка
