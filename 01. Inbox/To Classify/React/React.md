# React

## Краткое определение области

Обзорная заметка верхнего уровня для ветки `React` как компонентной UI-библиотеки и связанной с ней модели описания интерфейса, обновлений и повторных рендеров.

## Что входит в эту тему

- `[[JSX]]`
- `[[React Components]]`
- `[[React Rendering Model]]`

## Как устроена ветка

- `React` служит корневым `overview` для всей React-ветки.
- Внутри него стоит держать только устойчивые обзорные кластеры и самостоятельные статьи.
- Более узкие темы, вроде отдельных hooks или частных механизмов reconciliation, не должны подвешиваться прямо к корню без промежуточного обзорного узла.

## Рекомендуемый маршрут чтения

1. Начать с `React`.
2. Затем перейти к `[[JSX]]` и `[[React Components]]`.
3. После этого читать `[[React Rendering Model]]` как обзор обновлений, reconciliation и повторных рендеров.
