# Kendall Notation

## Краткое определение

`Kendall Notation` — это компактный способ обозначать класс системы обслуживания через свойства входящего потока, времени обслуживания и числа обслуживающих каналов.

## Основная идея или механизм

Запись вида `M/M/1` или `M/M/c` позволяет быстро зафиксировать, какие распределения предполагаются для поступления и обслуживания, а также сколько независимых обработчиков участвует в системе. Это дает общий язык для сравнения моделей без разворачивания длинного текстового описания.

## Границы темы

- Входит: сама схема обозначений и чтение типовых форм вроде `M/M/1`, `M/M/c`, `M/G/1`.
- Не входит: полное решение каждой модели и численные формулы для всех случаев.

## Примеры, случаи или следствия

Модель `M/M/1` полезна как минимальная аналитическая рамка для разговора о системе с одной очередью и одним обслуживающим каналом. Модель `M/M/c` уже лучше подходит для серверов с несколькими worker-потоками или параллельными обработчиками.
