# Vim Command-line Mode

## Краткое определение

`Vim Command-line Mode` — это статья о режиме Vim, в котором пользователь вводит `:`-команды, поисковые выражения, `?`-поиск и внешние filter-команды.

## Основная идея или механизм

Command-line mode в Vim нужен не для обычного текстового ввода, а для краткой строковой команды, которую редактор затем интерпретирует как `Ex`-инструкцию, search-pattern или shell-filter.

## Границы темы

- В тему входит сам режим ввода в нижней строке интерфейса и его роль как точки входа в `Ex`-команды и поиск.
- Рядом находится `[[Vim Ex Commands]]`, но она описывает уже семейство самих команд, а не режим, через который они вводятся.

## Примеры, случаи или следствия

Переход в `:`-строку показывает, что часть команд в Vim выражается не отдельными нажатиями в normal grammar, а короткими строковыми инструкциями вроде записи файла, поиска с диапазоном или пакетной замены.
