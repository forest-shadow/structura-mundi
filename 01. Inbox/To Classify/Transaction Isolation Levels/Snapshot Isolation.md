# Snapshot Isolation

## Краткое определение

Каноническая статья для модели изоляции, основанной на снимках состояния, которая часто используется в современных СУБД как практический компромисс.

## Основная идея или механизм

Нужно показать, как чтение из снимка меняет конкурентное поведение системы и почему эта модель важна для практического понимания `MVCC`-подобных систем.

## Границы темы

- Где `Snapshot Isolation` совпадает по интуиции с классическими уровнями изоляции.
- Где она ведет себя иначе и требует отдельного рассмотрения.
