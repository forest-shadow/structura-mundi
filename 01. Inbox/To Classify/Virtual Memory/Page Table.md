# Page Table

## Краткое определение

`Page Table` — это структура данных, которая хранит отображение виртуальных страниц процесса в физические фреймы и сопутствующие биты состояния.

## Почему тема находится в ветке Paging

Эта заметка относится именно к механике paging, потому что page table является центральной структурой перевода адресов, а не общей метафорой virtual memory.

## Основная идея или механизм

- виртуальный адрес разбивается на номер страницы и смещение;
- номер страницы используется для поиска записи в page table;
- запись сообщает, где находится соответствующий фрейм и в каком состоянии он доступен.

## Границы темы

- Входит: роль page table в address translation.
- Не входит: полный разбор всех архитектурных вариантов таблиц страниц и всех схем page replacement.

## Примеры, случаи или следствия

Эта заметка должна показать, почему без page table виртуальная адресация не может быть реализована как управляемое и проверяемое отображение.
