# Defer Script

## Краткое определение

Каноническая статья про атрибут `defer` у `<script>`, который откладывает исполнение скрипта до завершения парсинга документа.

## Основная идея или механизм

Эта заметка должна объяснять, как `defer` влияет на загрузку, порядок исполнения и взаимодействие со структурой HTML-документа.

## Границы темы

- Входит: поведение `defer` и его последствия для загрузки страницы.
- Не входит: общий разбор всех resource hints.
