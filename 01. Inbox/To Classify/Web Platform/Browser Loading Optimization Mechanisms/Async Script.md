# Async Script

## Краткое определение

Каноническая статья про атрибут `async` у `<script>`, который позволяет браузеру загружать внешний скрипт параллельно с парсингом HTML и исполнять его сразу после готовности.

## Основная идея или механизм

`async` уменьшает блокировку на этапе загрузки, но не гарантирует удобный порядок исполнения относительно других скриптов. Поэтому он лучше подходит для независимых сценариев, чем для основного прикладного кода страницы.

## Границы темы

- Входит: поведение `async` при загрузке и исполнении external scripts.
- Не входит: общий разбор всех browser loading optimization mechanisms.

Обычно полезен для аналитики, счётчиков и других независимых скриптов, которым не нужен строго предсказуемый момент исполнения относительно основного приложения.
