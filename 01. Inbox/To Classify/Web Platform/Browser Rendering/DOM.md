# DOM

## Краткое определение

Каноническая статья про `DOM` как внутреннюю объектную модель документа, с которой браузер и JavaScript работают при разборе и обновлении HTML-структуры страницы.

## Основная идея или механизм

Эта заметка должна объяснять, что `DOM` представляет документ как дерево узлов и служит одной из входных структур для последующего browser rendering.

## Границы темы

- Входит: структура документа, узлы, дерево, связь с HTML-парсингом и обновлениями страницы.
- Не входит: полный разбор всех стадий browser rendering pipeline.
